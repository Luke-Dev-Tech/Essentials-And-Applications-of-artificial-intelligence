🌍 Project Overview

The AI Recycling Trash Classifier is a group project designed to improve waste sorting using Artificial Intelligence. The system allows users to upload an image of a waste item and receive a prediction of the correct recycling category. The goal is to support sustainable waste management and reduce incorrect waste disposal.

🤖 How The AI Works

The system uses an image classification model trained on waste datasets. When a user uploads an image, the model analyzes visual features and predicts the category of waste such as plastic, paper, organic, or hazardous material.
